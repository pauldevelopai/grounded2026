# — Structural / rhythm tells —
# (These are described in prose because they're patterns, not phrases. The
#  editor pass is told to hunt them; the regexes below catch the mechanical ones.)
#
# * Rule-of-three lists ("faster, cheaper, and smarter") used as filler.
# * Every paragraph opening with a hedge ("While...", "Although...", "Though...").
# * Rhetorical questions used as section transitions.
# * Sentence fragments as aphoristic one-liners stacked back to back.
# * Symmetrical "It's not X, it's Y" constructions.

# — Regex tells (JS, matched gi) —
# Em-dash asides. Paul uses parentheses / ellipses, never the em-dash
# (7 in 78k corpus words). This is the single loudest tell in his register.

# " - not X but Y - " symmetrical antithesis
re:\bnot only\b[^.?!]*\bbut also\b
